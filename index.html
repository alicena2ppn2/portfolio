PROJECT 02

Explainable AI · Churn Prediction · Retention Strategy Dashboard

Retention 전략 수립 가능한 churn 예측 + XAI 원인 분석 시스템 구축

Readme 초안 작성 · md 파일 작성 · Streamlit 구현 · UI 개선 · 모델 코드 분석

SaaS 기업은 고객 이탈이 발생한 후에야 원인을 파악하는 경우가 많아, 선제적인 retention 전략 수립이 어렵습니다. 단순히 "이탈할 것 같다"는 예측만으로는 실무에서 활용하기 어렵고, 담당자가 납득할 수 있는 근거와 구체적인 개입 전략이 함께 필요하다고 판단했습니다. 이를 해결하기 위해 XAI 기반 이탈 예측 모델을 설계하고, 원인 분석부터 retention 액션까지 연결되는 대시보드를 구축했습니다.

accounts, subscriptions, feature_usage 등 다중 테이블을 account 단위로 통합하고, ML(Logistic Regression, Random Forest)과 DL(MLP) 모델을 비교 실험했습니다. 단순 정확도가 아닌 운영 threshold tuning(0.45)으로 실무 적용성을 높이고, SHAP 기반 XAI로 이탈 원인을 설명 가능한 형태로 시각화했습니다. 최종적으로 고객 유지 전략까지 연결되는 Streamlit 대시보드를 배포했습니다.

논문 · 저작권

Ultra-High Reliable Multi-AP Cooperation Utilizing Securely Compressed Channel Status Information

Wi-Fi 8의 Multi-AP cooperation 환경에서 채널 상태 정보 오버헤드와 보안 위협을 동시에 해결하는 BC-SCMM(Bitmap Compression-based Secure Channel Management Mechanism)을 제안했습니다.

해시 기반 비트맵 압축·암호화를 통해 기존 대비 latency 74.12% 감소, throughput 232.49% 향상, energy consumption 105.19% 감소를 실험으로 검증했습니다.

대외활동

8개월간 KIRD 리얼챌린지 프로그램에 참가하여 실제 연구과제 기획부터 최종 발표까지 전 과정을 경험했습니다.

연구 주제
공공 와이파이 디지털 면역체계 — 디지털 취약계층을 위한 공공 와이파이 자동복구 기술

연구 배경 및 문제 정의
디지털 취약계층(기초생활수급자, 노년층 등) 중 40%가 통신비 부담으로 인터넷을 이용하지 못하는 상황에서, 대안으로 제시된 공공 와이파이는 관리시스템 미구축으로 장애 건수가 2019년 24건에서 2022년 417건으로 1,637% 급증하는 문제가 있었습니다. 장애의 73.6%가 소프트웨어 문제로, 리부팅 등 간단한 조치만으로 해결 가능하지만 자동화된 모니터링 체계가 없었습니다.

제안 기술
NIA(한국지능정보사회진흥원) 및 연구팀과의 이해관계자 인터뷰를 바탕으로 두 가지 핵심 기술을 설계했습니다. 첫째, 머신러닝 기반 Passive WiFi 트래픽 모니터링으로 시간대별 트래픽 통계치를 분석해 과소·과다 트래픽 패턴을 자동 탐지합니다. 둘째, Active 모니터링 메커니즘으로 이상 감지 시 리부팅·릴레이 등 자동 복구를 수행하는 2-step 장애 대응 체계를 제안했습니다.

기대 효과
추가 장비 설치 없이 저비용·고효율 모니터링만으로 안정적인 통신 서비스 제공이 가능하며, 재난 상황에서도 끊김 없는 통신 확보 및 디지털 정보격차 해소에 기여할 것으로 분석했습니다.
